Fiori Tracker

Documentation automated

Fiori Tracker is a documentation tool.

Replaces multiple spreadsheets

Guides project team members through the process of documenting requirements of SAP Fiori Launchpad configuration.

Makes sure that outcome is consistent an inline with agreed standards.

The tool helps to:

  • Avoid duplication
  • Follow naming conventions

Replaces multiple spreadsheets

One, easy to access, consistent, reference point

Your Fiori apps records stay linked with:

  • roles
  • catalogs
  • test users
  • change requests
  • comments
  • issue requests
  • change history records
  • As-is information

Additional tools: Fiori Apps' Usage

  • Monitoring SAP Fiori apps usage (not available in standard)
    • Statistics also available in „As-is” section of Applications details

Use cases:

  • Report on apps setup progress
  • Test users during implemetation
  • Usage data while testing
  • Scope creep control
