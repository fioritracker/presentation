Fiori Tracker

Documentation automated

Fiori Tracker is a documentation tool.

Automates documentation preparation and enables comparison with actual system configuration.

Replaces multiple spreadsheets

Guides project team members through the process of documenting requirements of SAP Fiori Launchpad configuration.

Makes sure that outcome is consistent an inline with agreed standards.

The tool helps to:

  • Avoid duplication
  • Follow naming conventions
